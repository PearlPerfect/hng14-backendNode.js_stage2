Natural Language Parser Reference
Gender keywords
male, males, man, men → gender=male
female, females, woman, women → gender=female
"male and female" → no gender filter
Age keywords
young → min_age=16, max_age=24
above/over/older than N → min_age=N
below/under/younger than N → max_age=N
between N and M → min_age=N, max_age=M
Age group keywords
child, children, kids → age_group=child
teenager, teen, teens → age_group=teenager
adult, adults → age_group=adult
senior, seniors, elderly → age_group=senior
Country keywords
from/in + country name → country_id=ISO
"nigeria" → NG, "ghana" → GH
"south africa" → ZA, "kenya" → KE
2-letter ISO also supported
